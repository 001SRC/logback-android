logback-android by tony19

The reliable, generic, fast and flexible logging framework for Java on Android.

Logback-Android brings the power of Logback to Android. Logback is a reliable, generic, fast, and flexible logging library for Java applications written by the creator of the popular (but now defunct) Apache log4j project. Logback-Android provides a richer API than android.util.Log (including automatic log file compression). Additionally, Logback-Android together with SLF4J allows for greater logging flexibility and portability across Java platforms. The current version is 1.0.0-2.

Dependencies

SLF4J 1.6.4

Install

Add the libraries for Logback and SFL4J to your project: * logback-android-x.x.x-y.jar * slf4j-api-x.x.x.jar Do not include slf4j-android.jar.
